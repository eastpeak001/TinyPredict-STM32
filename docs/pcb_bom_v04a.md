# TinyPredict-STM32 V0.4A PCB BOM

V0.4A 是基于 Blue Pill / STM32F103C8T6 开发板的扩展底板，不直接焊接 STM32 芯片。PCB 主要用于替代面包板和杜邦线，连接 MPU6050、OLED、USB-TTL、SWD、电源、LED 和蜂鸣器接口。

## BOM 使用说明

- V0.4A 的目标是降低第一版 PCB 风险，因此优先使用已经验证过的开发板和外设模块。
- MPU6050 和 OLED 均通过 4Pin 接口接入，方便更换模块和调整安装位置。
- USB-TTL 不集成到 PCB，只保留 USART1 调试接口。
- 蜂鸣器为预留功能，第一版可只验证接口，不要求固件立即支持。
- 风扇电机不要从本 PCB 取电，PCB 只负责监测、显示和通信。
